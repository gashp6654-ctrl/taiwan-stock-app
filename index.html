📈 MA5｜1週均線
計算過去 5個交易日的平均價。短線交易、當沖、隔日沖常用。適合判斷短期市場情緒與大戶博心理戰，但不適合拿來判別公司長期好壞。
📈 MA20｜月均線（成本防線）
計算過去 20個交易日的平均價。有些人視為「成本防線」，股價維持在月線之上代表中期趨勢未變，法人可能持續鎖籌碼。帶量跌破 MA20，可能代表大戶撤退，是中期投資、波段操作者的重要參考線。
📈 MA100｜20週均線（終極防線）
計算過去 100個交易日的平均價。過濾所有短中期雜訊後的「真實趨勢」。長線投資、價值投資、大波段操作者的終極防線，通常用於判斷一家公司長期趨勢與法人態度。

📊 成交量｜分辨真假訊號的關鍵
一、價量關係基礎規則
✅ 價漲量增（最健康）
市場高度認同，買盤願意加價進場，上漲趨勢通常延續。
⚠️ 價漲量縮（背離）
追價力道薄弱，可能是散戶在推或主力縮手，隨時可能反轉回跌。
✅ 價跌量縮（良性修正）
賣壓不重，持股者惜售，通常是健康盤整，未來有機會重啟漲勢。
🚨 價跌量增（恐慌殺盤）
市場信心崩潰，大家不計代價出脫持股，股價容易繼續探底。
二、結合均線的關鍵徵兆
✅ 帶量突破（真實）
站上 MA20 或 MA100 時成交量達平時 1.5～2倍以上，代表大戶法人出錢出力強攻，是很好的買進訊號。
⚠️ 無量突破（假突破）
股價勉強站上均線但量能平平，通常很快就跌回來套牢追高的人。
三、特殊警訊：爆量
📍 低檔爆量（見底訊號）
股價跌很久後在 MA100 下方突然驚人巨量，通常是主力低位吃貨，可能是長線反轉的前兆。
📍 高檔爆量（見頂訊號）
沿著 MA5 連漲好幾天後突現天量長紅或長黑，往往是「利多出盡」，大戶趁熱鬧把貨倒給散戶，要小心大回檔。

📈 季營收 + 季毛利率｜質與量同步分析
你觀察過「營收創新高、股價卻不漲」的情況嗎？這通常是毛利率出問題的警訊。毛利率越高，代表公司扣除直接成本後能留下的錢越多，幫你判斷公司是「表面訂單接不完根本沒賺錢」還是「質與量同步成長」。
四種情境綜合分析
✅ 營收升 + 毛利升（最理想）
產品供不應求，公司甚至有漲價能力，或新產品毛利更高。
⚠️ 營收升 + 毛利降（需警戒）
可能採「降價競爭」搶市佔率，或原物料成本大幅上升無法轉嫁。
🔄 營收降 + 毛利升（轉型期）
公司可能精簡產品線，放棄低毛利訂單，專注高價值產品。
🚨 營收降 + 毛利降（衰退中）
產業可能進入寒冬或競爭力喪失，產品賣不動且必須降價出清。
